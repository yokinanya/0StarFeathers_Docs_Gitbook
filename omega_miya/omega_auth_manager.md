# 授权管理
> 插件特殊权限授权管理
> 仅限管理员使用

---
## 使用方法
```
#OmegaAuth [授权操作] [授权对象ID] [插件名称] [权限节点]
#OmegaAuth [allow|deny] [插件名称] [权限节点]
#OmegaAuth [list]
可用授权操作:
allow: 允许会话所在群组/频道/用户
deny: 禁止会话所在群组/频道/用户
list: 列出会话所在群组/频道/用户已配置的权限节点
custom_allow: 允许指定群组/频道/用户
custom_deny: 禁止指定群组/频道/用户
```
